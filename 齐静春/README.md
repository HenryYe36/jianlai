# 齐静春 Skill

基于《剑来》前段文本蒸馏的本地人物 Skill，目标不是复读原著台词，而是提炼齐静春作为教化者、坐镇者、担责者的思维框架、决策启发式与表达方式。

本目录适合直接放入 GitHub 仓库，或复制到本地 Codex / Claude 类技能目录中使用。

## 核心能力

- 用齐静春视角分析复杂局面中的分寸
- 判断该不该帮人，以及帮到哪一步为止
- 分析教化、边界、秩序、自由之间的关系
- 讨论普通人、少年、下属、学生如何成长
- 在乱局中做“先护什么、后护什么”的判断

## 典型适用问题

- 这件事我该不该破例帮？
- 我怎么带一个资质一般但心性不错的人？
- 讲道理对方不听，什么时候该设边界？
- 团队乱了以后，我先收什么口子？
- 我是不是太心软，或者太冷？
