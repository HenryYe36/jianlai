# 如何使用阿良 Skill

这份文档说明如何在本地使用 `a-liang-perspective`。

## 1. 安装方式

如果你的工具支持本地 skills 目录，通常只需要把 `a-liang-perspective` 整个目录复制到对应 skills 目录即可。

例如：

```bash
cp -R a-liang-perspective ~/.codex/skills/
```

## 2. 直接使用

安装后，可直接用这些句式触发：

- 用阿良的视角看看这件事
- 阿良会怎么劝我
- 切换到阿良模式
- 怎么把话说重一点，但别把人说垮
- 做好人太累了，用阿良的方式看看

## 3. 它会做什么

激活后，这个 Skill 通常会按这个顺序分析：

1. 先看这个人眼下缺胆气还是缺分寸
2. 再判断他更需要鼓劲还是更需要被点醒
3. 先把门打开
4. 再把真正的话说出来
5. 最后落到“下一步怎么走”

## 4. 最适合的场景

- 提气
- 带年轻人
- 远行与成长
- 做好人为何辛苦
- 练拳练剑和长期积累
- 怎么把话讲得能进心里

## 5. 不适合的场景

这份 Skill 不适合：

- 医疗、法律、税务等专业结论
- 纯技术问题
- 需要全书后期阿良完整细节的考据

## 6. 推荐提问模板

### 模板一：提气

```text
我现在有点灰心。请用阿良的方式跟我说说这件事。
```

### 模板二：带人

```text
如果阿良来带这个年轻人，他会怎么说、怎么做？
```

### 模板三：做好人太累

```text
做好人太累了，用阿良的视角看，我该怎么想、怎么做？
```

### 模板四：长期积累

```text
练一件长期的事，我总觉得太慢了。请用阿良的方式点拨我。
```
