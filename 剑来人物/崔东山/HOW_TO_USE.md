# 如何使用崔东山 Skill

这份文档说明如何在本地使用 `cui-dongshan-perspective`。

## 1. 安装方式

如果你的工具支持本地 skills 目录，通常只需要把 `cui-dongshan-perspective` 整个目录复制到对应 skills 目录即可。

例如：

```bash
cp -R cui-dongshan-perspective ~/.codex/skills/
```

## 2. 直接使用

安装后，可直接用这些句式触发：

- 用崔东山的视角看看这件事
- 崔东山会怎么看这个人
- 切换到崔东山模式
- 请用崔东山的方式帮我拆局
- 如果我是落魄山的二号位，该怎么做

## 3. 它会做什么

激活后，这个 Skill 通常会按这个顺序分析：

1. 先拆棋盘层次
2. 再看每个人站在哪一格
3. 再判断谁可用、谁可防、谁该踢出去
4. 再决定这件事说几分、试几分、动几分
5. 最后给出站位和动作建议

## 4. 最适合的场景

- 谋局
- 试探
- 识人
- 忠诚与站队
- 二号位思维
- 守门与补位
- 说怪话但不失分寸

## 5. 不适合的场景

这份 Skill 不适合：

- 纯情绪安抚
- 医疗、法律、税务等专业结论
- 需要低刺激、低锋芒表达的陪伴型场景

## 6. 推荐提问模板

### 模板一：拆局

```text
请用崔东山的方式分析，这盘局的真正棋盘有几层？
```

### 模板二：识人

```text
这个人到底是真忠诚，还是只是嘴上好听？请用崔东山的角度看。
```

### 模板三：二号位

```text
如果我是一个团队里的二号位，怎么做才算真正补位，而不是瞎忙？
```

### 模板四：怪话

```text
我想试探对方，又不想直接摊牌。崔东山会怎么说、怎么做？
```
