# 如何使用文圣 Skill

这份文档说明如何在本地使用 `wen-sheng-perspective`。

## 1. 安装方式

如果你的工具支持本地 skills 目录，通常只需要把 `wen-sheng-perspective` 整个目录复制到对应 skills 目录即可。

例如：

```bash
cp -R wen-sheng-perspective ~/.codex/skills/
```

## 2. 直接使用

安装后，可直接用这些句式触发：

- 用文圣的视角看看这件事
- 老秀才会怎么看
- 切换到文圣模式
- 请用老秀才的方式帮我讲讲这件事的顺序
- 从先生的角度看，我该怎么带这个学生

## 3. 它会做什么

激活后，这个 Skill 通常会按这个顺序分析：

1. 先捋顺序有没有乱
2. 再分这是书上的问题，还是活人的问题
3. 再看牵涉到谁，是学生、朋友、自己人还是陌生人
4. 最后决定是点醒、设边界，还是干脆起而行之

## 4. 最适合的场景

- 教化
- 带学生
- 读书与做人
- 次序与轻重
- 规矩与人情
- 学问如何落地

## 5. 不适合的场景

这份 Skill 不适合：

- 纯技术细节问题
- 医疗、法律、税务等专业结论
- 完全不需要“讲活道理”的纯数据型决策

## 6. 推荐提问模板

### 模板一：顺序

```text
请用文圣的方式帮我捋一捋，这件事到底先做什么、后做什么。
```

### 模板二：带学生

```text
如果从先生的角度看，我该怎么带这个学生，既不误人，也不把他带死？
```

### 模板三：书上与现实

```text
书上的道理和眼前活人的难处冲突了。请用文圣的角度帮我拆。
```

### 模板四：朋友与人情

```text
文圣会怎么看朋友、人情和读书之间的关系？
```
