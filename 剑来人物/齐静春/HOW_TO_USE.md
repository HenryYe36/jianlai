# 如何使用齐静春 Skill

这份文档说明如何在本地使用 `qi-jingchun-perspective`，以及如何触发它。

## 1. 安装方式

如果你的工具支持本地 skills 目录，通常只需要把 `qi-jingchun-perspective` 整个目录复制到对应 skills 目录即可。

例如：

```bash
cp -R qi-jingchun-perspective ~/.codex/skills/
```

如果你使用的是项目级 skills，也可以直接把目录放在项目里的 `.claude/skills/` 或同类位置。

## 2. 直接使用

安装完成后，在对话里用这些句式触发即可：

- 用齐静春的视角看看这件事
- 齐静春会怎么看这个人
- 切换到齐先生模式
- 我该不该帮这个人，用齐静春的方式分析
- 这件事我想讲道理但怕越界，按齐静春的路数怎么看

## 3. 它会做什么

激活后，Skill 通常会按这个顺序分析：

1. 先看局势，而不是先看情绪
2. 先分清该护什么，再分清该护谁
3. 区分“能被点醒的人”和“只能被设边界的人”
4. 判断这次帮忙是“扶一把”还是“替人背命”
5. 给一个克制、可执行、不越俎代庖的建议

## 4. 最适合的场景

- 带人
- 教学
- 团队管理
- 家庭冲突
- 合作边界
- 是否破例
- 长期成长
- 混乱局面中的轻重缓急

## 5. 不适合的场景

这份 Skill 不适合直接输出专业结论的领域，例如：

- 医疗诊断
- 法律意见
- 税务建议
- 需要精确数据的金融判断

在这些问题上，它最多适合作为“分寸和边界”的辅助视角，不应替代专业判断。

## 6. 推荐提问模板

### 模板一：是否帮人

```text
用齐静春的视角看，这个人值得我帮吗？我帮到哪一步为止？
```

### 模板二：带人

```text
如果齐先生来带这个下属/学生，他会怎么做？
```

### 模板三：乱局判断

```text
局面已经乱了，我先护什么，后护什么？请用齐静春的方式分析。
```

### 模板四：边界

```text
我怕自己太心软，或者太冷。齐静春会怎么拿捏这个分寸？
```

## 7. 目录内文件说明

- `SKILL.md`：主 Skill 文件，定义角色、工作流、模型、启发式、表达方式
- `references/research/*.md`：调研与蒸馏依据
- `references/sources/books/剑来-齐静春前段语料说明.md`：原始素材范围说明
- `examples.md`：真实风格示例问答，便于先看输出效果

## 8. 如果你要继续扩展

推荐的扩展方向：

1. 增补齐静春与崔瀺、先生文脉相关的后段材料
2. 把齐静春常用判断句进一步整理成 prompt 模板
3. 再补一个 `examples.md`，放真实问答示例
