# 陈平安 Skill

基于《剑来》原文本地语料蒸馏的人物 Skill，重点不是复读台词，而是提炼陈平安在不同阶段体现出来的：

- 做人分寸
- 扛事方式
- 守诺逻辑
- 吃苦与积累方法
- 带人和立规矩的路数

本目录适合直接放入 GitHub 仓库，或复制到本地 Codex / Claude 类技能目录中使用。

## 核心能力

- 用陈平安视角分析承诺、恩怨、人情和责任
- 讨论普通人如何靠笨功夫、吃苦和年头往前走
- 判断一件事值不值得扛、该不该扛到底
- 分析带人、立规矩、建团队、做长期经营
- 在现实账目和道理之间找平衡

## 典型适用问题

- 这件事我答应了，现在很重，还该不该继续扛？
- 我是不是把太多责任揽到自己身上了？
- 普通人怎么靠长期积累翻身？
- 我怎么带一个徒弟、晚辈或者团队？
- 人情、钱、规矩、道理混在一起时，先看什么？
