# 06 人物时间线

## 关键节点

| 阶段 | 事件 | 提炼意义 |
|------|------|---------|
| 少年贫寒期 | 父母离世、龙窑断生计 | 建立“先活下去”的现实底色 |
| 学徒与苦活期 | 拉坯、跑山、送信挣钱 | 形成笨功夫、自立和耐性 |
| 齐静春介入期 | 被点破身世、承接道理 | 开始把“活着”升级成“站得住” |
| 宁姚养伤期 | 开门救人、抓药照料 | 责任意识由自保扩展到担当 |
| 练拳启蒙期 | 学走桩、学长期坚持 | 把修行理解为一遍遍重复 |
| 离镇准备期 | 接住托付，准备走出去 | 从小镇伦理进入更大天地 |
| 落魄山初建期 | 买下落魄山、安排山神、竹楼、黑蟒与山路 | 从个人生存转入“经营一个地方” |
| 远游磨砺期 | 藕花福地、书简湖线索、一路见世道 | 开始学会把善意与锋芒同时留在身上 |
| 剑气长城期 | 接触隐官体系、真正见识大战与规矩 | 说话做事都更懂分寸、代价与权柄 |
| 返乡与带人期 | 回落魄山、带裴钱与山上众人 | 从独行者转为能定规矩、能搭秩序的人 |

## 时间线结论

陈平安的成长不是“突然开悟”，而是：
- 苦难打底
- 他人点醒
- 自己兑现
- 后期再加上一层：把个人道理，变成能安顿他人的做法

前期的陈平安，重点是自己不倒。
后期的陈平安，重点是自己站住了，还要让别人也有地方站。
