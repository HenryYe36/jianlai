# 如何使用陈平安 Skill

这份文档说明如何在本地使用 `chen-pingan-perspective`，以及它最适合回答什么问题。

## 1. 安装方式

如果你的工具支持本地 skills 目录，通常只需要把 `chen-pingan-perspective` 整个目录复制到对应 skills 目录即可。

例如：

```bash
cp -R chen-pingan-perspective ~/.codex/skills/
```

如果你使用项目级 skills，也可以直接把目录放进项目里的 `.claude/skills/` 或同类位置。

## 2. 直接使用

安装后，可直接用这些句式触发：

- 用陈平安的视角看看这件事
- 陈平安会怎么看这个人
- 切换到陈平安模式
- 如果是陈平安，会不会答应这件事
- 用陈平安的方式分析我该不该扛这个责任

## 3. 它会做什么

激活后，这个 Skill 通常会按这个顺序分析：

1. 先看我答应过什么
2. 再看这件事里谁该担责
3. 再看眼前最急的是什么
4. 再算清楚代价是自己扛，还是别人替我扛
5. 最后判断这件事做完以后，人还能不能站得住

## 4. 最适合的场景

- 承诺与责任
- 朋友、家人、同伴之间的人情问题
- 普通人的长期成长
- 吃苦、熬、重复积累
- 建团队、带徒弟、立规矩
- 钱、人情、道理交织的现实问题

## 5. 不适合的场景

这份 Skill 不适合直接给出专业结论的领域，例如：

- 医疗诊断
- 法律意见
- 税务和合规结论
- 依赖实时外部数据的金融分析

它更适合做“做人做事的判断辅助”，不替代专业判断。

## 6. 推荐提问模板

### 模板一：承诺与责任

```text
用陈平安的视角看，这件事我既然答应了，现在还该不该继续扛？
```

### 模板二：普通人成长

```text
如果按陈平安的路数，一个普通人应该怎么靠长期积累往前走？
```

### 模板三：带人

```text
如果陈平安来带这个徒弟/晚辈/团队，他会怎么做？
```

### 模板四：现实决策

```text
这件事里人情、钱、道理混在一起了。请用陈平安的方式帮我分清轻重。
```

## 7. 目录内文件说明

- `SKILL.md`：主 Skill 文件，定义角色、工作流、模型、启发式与表达方式
- `references/research/*.md`：调研与蒸馏依据
- `references/sources/books/剑来-source-notes.md`：原始素材说明
- `examples.md`：示例问答，便于先看输出效果

## 8. 如果你要继续扩展

推荐方向：

1. 按不同阶段拆分“少年陈平安 / 落魄山陈平安 / 后期带人模式”
2. 补更多“说怪话”与“护短但不失规矩”的例子
3. 单独整理“带裴钱”和“落魄山立规矩”两条子专题
