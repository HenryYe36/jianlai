# 如何使用崔瀺 Skill

这份文档说明如何在本地使用 `cui-can-perspective`，以及它最适合回答什么问题。

## 1. 安装方式

如果你的工具支持本地 skills 目录，通常只需要把 `cui-can-perspective` 整个目录复制到对应 skills 目录即可。

例如：

```bash
cp -R cui-can-perspective ~/.codex/skills/
```

如果你使用项目级 skills，也可以直接放到 `.claude/skills/` 或同类位置。

## 2. 直接使用

安装后，可直接用这些句式触发：

- 用崔瀺的视角看看这盘局
- 崔瀺会怎么看这件事
- 切换到绣虎模式
- 如果从治国和谋局的角度看，该怎么落子
- 请用崔瀺的方式帮我做一个冷决策

## 3. 它会做什么

激活后，这个 Skill 通常会按这个顺序分析：

1. 先拆结构
2. 再找主线变量
3. 再判断谁在逼你动、谁在误判
4. 再算清此时动与不动的代价
5. 最后给出落子建议

## 4. 最适合的场景

- 战略分析
- 权力和组织治理
- 大项目取舍
- 时机判断
- 高代价但不得不做的决策
- 如何在复杂局里控制信息披露

## 5. 不适合的场景

这份 Skill 不适合：

- 纯感情安抚
- 需要温和陪伴式回应的问题
- 医疗、法律、税务等专业结论
- 需要全书后期崔瀺完整信息才能下定论的细节型考据

## 6. 推荐提问模板

### 模板一：谋局

```text
请用崔瀺的视角分析，这盘局真正该落子的地方在哪里？
```

### 模板二：组织治理

```text
如果按崔瀺的方式，这个团队乱局应该先收什么口子？
```

### 模板三：冷决策

```text
我需要为了全局做一个不讨喜的决定。请用崔瀺的路数帮我判断。
```

### 模板四：时机

```text
对手现在逼我表态，我该现在动，还是继续忍？请用绣虎的方式看。
```

## 7. 目录内文件说明

- `SKILL.md`：主 Skill 文件
- `references/research/*.md`：调研与蒸馏依据
- `references/sources/books/剑来-崔瀺当前语料说明.md`：素材边界说明
- `examples.md`：示例问答
