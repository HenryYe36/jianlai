# 剑来人物 Skills

这是一个把《剑来》人物蒸馏成可运行思维 Skill 的仓库。

目标不是做人物介绍，也不是摘抄原著台词，而是尽量把人物在文本中体现出来的：

- 思维框架
- 决策启发式
- 表达方式
- 反模式
- 适用场景

整理成可复用的本地 Skill。

## 如何使用

如果你只是想直接用某个人物 Skill：

1. 打开对应人物目录
2. 阅读该目录下的 `README.md` 和 `HOW_TO_USE.md`
3. 把人物目录中的 `*-perspective/` 复制到本地 skills 目录
4. 在对话里用该人物名字或视角句式触发

例如齐静春：

```text
用齐静春的视角看看这件事
齐先生会怎么看这个人
这件事我该不该破例帮，用齐静春的方式分析
```

## 设计原则

这个仓库的每个人物 Skill 尽量满足：

1. 自包含  
复制整个人物目录即可使用，不依赖仓库外部文件。

2. 本地语料优先  
如果有明确文本素材，优先从文本里蒸馏，而不是凭印象拼接。

3. 不复读人物口头禅  
重点提炼“他如何判断”，而不是只模仿“他说话像不像”。

4. 有边界说明  
明确 Skill 擅长什么，不擅长什么。
